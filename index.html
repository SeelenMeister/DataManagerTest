Hola! bienvenido/a a mi App de prueba de un administrador de datos (Data Manager).

Esta interfaz está orientada a los "Operadores". Desde aquí se reciben los datos que se utilizaran para realizar las llamadas. Luego a través de la opcion de "tabulacion" se clasifica el dato para una mejor gestion y seguimiento de la base de datos.

Solamente despues de "tabular/clasificar" el dato, se obtiene uno nuevo.

El boton "copiar" sirve para marcar el numero de telefono sin necesidad del teclado (tambien se puede adaptar la funcion para predeterminar un programa de discado).

Se puede seleccionar el numero de telefono para modificarlo antes de copiarlo (no afecta al dato en la tabla original)

En la WIKI hay Información mucho más detallada. Clickeame para visitarla!.
